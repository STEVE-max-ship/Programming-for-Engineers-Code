Removing Noise from an Image

This is a modification of the exam problem.

In this modification, only Type 1 noise is processed.

For each input, remove from the matrix only the noise blobs of type type 1.
Do not process or evaluate any of noise blobs of type 2,3, or 4.
Also, on the last output line print only the number of the removed noise blobs of type 1. See examples below.
The public data at the bottom of this page are modified accordingly.
