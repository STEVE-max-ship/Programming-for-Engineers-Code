PGE Exam Problem

Z-bent Paths in an Array

Let us consider a rectangular array with R rows and C columns, containing numerical values.
A path is a sequence of cells in the array, in which each next cell (with the exception of the first cell) is adjacent to the previous cell in the path.
In this problem, we consider adjacency in only four principal directions: left, righ, up, down. Two cells which share only one corner (they are positioned diagonally with respect to each other) are not adjacent.

A z-bent path is a path with a distinct shape, position, contents of the path itself and its immediate surroundings. It is described below. (While reading the description, step by step, you may compare it with example Image 1.)
A z-bent path consists of three straight parts. The width of each part is exacly one cell.
The first part and the third part are horizontal, the middle part (the second part) is vertical.
The last cell in the first part is equal to the first cell in the second part and the last cell in the second part is equal to the first cell in the third part.
The first part starts at the left border of the array. The third part ends at the right border of the array.
No cell in any part touches the top border or the bottom border of the array. No cell in the central (second) part touches any border of the array.
The number of cells in the central part of the z-bent path is 4.
The first part is closer to the top border of the array then the third part.
All cells which are part of a z-bent path contain zeroes.
Each cell which is not on the z-bent path and which is adjacent to some cell in the z-bent path contains a positive value.
Any path or any sequence of cells which does not satisfy all conditions given above is not a z-bent path.

The coordinates of a z-bent path consist of two numbers. The first number is the index of the row in which the z-bent path starts, and the second number is the index of the column in which the central part is located.

The cells which are adjacent to the cells of z-bent paths are depicted with dark background and white font. Note that all those cells contain non-zero values. The Image illustrates Example 1 below.

The task

Print the coordinates of all z-bent paths in the given array.

Input

The first input line contains two positive integers R, C separated by space. They represent, in this order, the number of rows in the array and the number of columns in the array.
Next, there are R lines of input, each contains C values separated by spaces. Each line contains values of the corresponding array row. The size of the array is at most 1000 × 1000. The values in the array are non-negative integers less then 10.

Output

The output contains the coordinates of all z-bent paths in the given array, in the ascending order of the rows. Each output line contains the coordinates of one z-bent path, separated by space.
It is guaranteed that there is always at least one z-bent path in the input array.
